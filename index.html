Hi, we're Grapple.

We think everyone should be able to boulder, when it suits them. At Grapple we aim to solve some of the problems currently associated with bouldering to bring it to everyone.

Far Locations

Climbing centres are great, but the large area required for a climbing centre means they can't be situated in city centres. The fact that climbing centres can be difficult to reach may be ofputting to many climbers.

Fixed Routes

The routes available to climb at climbing centres are typically set by climbing centre staff, and can remain on a wall for up to a month depending on how frequently the staff update the routes. Once you solve as many routes as you can, climbers need to wait for route updates to try new problems.

Non-Automation

Changing climbing routes requires an experienced staff member to manually remove and replace each hold, iteratively testing the route as they go. This can consume a lot of time and manpower.

The Big Bouldering Wall, by Grapple.

Featuring an innovative mechanism for automated climbing hold movement, the Big Bouldering Wall fits all your climbing needs into a compact package, perfect for climbing where you want, when you want.

Climbing brought to you.

The Big Bouldering Wall can be set up in a nearby gym or even your home; measuring only 1.2 m by 2.4 m by 1 m, the compact build means you will always have access to climb where you want, when you want.

Your climb, your routes.

Climbing routes are usually set by climbing centre staff, with a wide variety of difficulties and styles.
But we think allowing you to create and choose your own climbing routes is a more valuable bouldering experience.

Routes that change for you.

The usual method of changing climbing routes requires a technician to manually remove and replace climbing holds. This not only takes up time and resources but also only allows for a limited set of hold placements.
Automation is the next big step to make your bouldering experience truly seamless.

Shared Climbing.

Bring a whole new twist to your climbing experience, by sharing with friends.

Website Creation Tools

This website was created using a HTML/CSS/JavaScript template available to download free from Bootstrap. Bootstrap templates are useful because most of the aesthetic and functional portions of the website have already been set up, so only the content needs to be altered to suit the specific application. The source code content of the website was modified using the Atom source code editor, and code changes were tracked using GitHub Desktop. The combination of Atom and GitHub desktop results in an efficient workflow that can edit and push local changes quicky.

How to Set Up a Domain Name

  • Step 1

    Choose a Domain

    Pick a catchy, memorable domain for your website that isn't already taken. You can check whether your ideal domain is available using services like GoDaddy and 123Reg.

  • Step 2

    Register & Buy the Domain

    You can purchase your domain an annual basis, with prices varying depending on what extension you want. For example, ".com" websites typically cost more than ".net". You can buy your domain for a maximum of 10 years in one go.

  • Step 3

    Pick a Web Host

    Once you have a domain, you will need server space to store all the files needed to make your website function. This is called Web Hostinng, and most domain registrars will include some form of web hosting along with registration.

  • Your website
    is now
    live!
